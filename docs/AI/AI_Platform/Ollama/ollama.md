- [Ollama 官网模型页面](https://ollama.com/search)

- 指定路径安装：`OllamaSetup.exe /DIR="d:\some\location"`

### 常用命令

- `ollama list` 查看下载模型
- `ollama ps` 查看运行部署模型
- `ollama show MODEL [flags]` 查看模型信息
- `ollama ollama run MODEL [PROMPT] [flags]` 启动运行模型
- `ollama stop MODEL [flags]` 停止运行模型
- `ollama rm MODEL [MODEL...] [flags]` 删除模型
- `ollama launch`
- `ollama serve [flags]` 启动ollama

- `ollama pull MODEL` 下载模型
- `ollama --version`
