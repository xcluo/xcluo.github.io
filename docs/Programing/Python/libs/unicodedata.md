### `category`

#### Other
- [Cc] Control
- [Cf] Format
- [Cn] Not Assigned (no characters in the file have this property)
- [Co] Private Use
- [Cs] Surrogate  


#### Letter
- [LC] Cased
- [Ll] Lowercase
- [Lu] Uppercase
- [Lm] Modifier
- [Lo] Other
- [Lt] Titlecase  


#### Mark
- [Mc] Spacing Combining
- [Me] Enclosing
- [Mn] Nonspacing  


#### Number
- [Nd] Decimal Digit
- [Nl] Letter
- [No] Other  


#### Punctuation
- [Pc] Connector
- [Pd] Dash
- [Pe] Close
- [Pf] Final quote (may behave like Ps or Pe depending on usage)
- [Pi] Initial quote (may behave like Ps or Pe depending on usage)
- [Po] Other
- [Ps] Open  


#### Sysbol
- [Sc] Punctuation
- [Sk] Modifier
- [Sm] Math
- [So] Other
- [Zl] Line
- [Zp] Paragraph
- [Zs] Space
