

### 信息查看
1. `docker --version`
2. `docker ps`，查看运行的 docker 容器信息
    - `-a`，显示（包括未运行的）所有 docker 容器信息 
5. `docker image`，列出所有镜像

### 启停、增删容器
1. `docker build`
3. `docker rm container_id_or_name`，删除运行指定 docker 容器
4. `docker run`
5. `docker stop container_id_or_name`，停止运行指定 docker 容器
6. `docker exec`

### 仓库操作 
1. `docker pull`
2. `docker push`
3. `docker login`
4. `docker logout`
5. `docker search`
6. `docker tag`
7. `docker inspect`
8. `docker rmi`
