### 文件操作

#### 文件内容查看

- 原始文本：`less/more`、`head/tail`
- 条件选择：[`awk`](file_related/awk.md)
- 内容过滤：`grep`
- 内容比较：[`diff`](file_related/diff)

#### 文件信息查看

- 信息查看：`ls`
- 内容统计：`wc`

#### 文件修改

- 信息修改：`mv`、`cp`、`rm`
- 内容合并：`cat`
- 内容拆分：[`split`](file_related/split)
- 内容去重：`sort` + `uniq`
- 内容替换：`tr`、`sed`
- 内容扰动：`shuf`
