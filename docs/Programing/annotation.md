#### C

- **CORS**: **C**ross-**O**rigin **R**esource **S**haring，跨域资源共享, 自定义某些白名单源可以跨源访问服务器资源，从而绕过同源策略

#### D

- **DI**：**D**ependence **I**njection，依赖注入，在参数传递至目标函数前，先进行前置注入函数的处理

#### I

- **IAM**：**I**dentity and **A**ccess **M**anagement，身份与访问管理

#### J

- **JTW**: **J**SON **W**eb **T**oken，用于身份验证和授权的token令牌

#### S

- **socket** 套接字，由IP地址和端口号组成，如 `127.0.0.1:8080`

- **SOP**: **S**ame-**O**rigin **P**olicy，同源（http/https + IP + 端口号）策略，要求源一致才可访问服务器资源，用于限制跨域请求，维护安全性
