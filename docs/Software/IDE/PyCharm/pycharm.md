项目内模块代码导入时，显示找不到文件

- 右键对应模块文件夹，选择`Mark Directory as → Source Root`

### 快捷键
#### Ctrl
- `Ctrl + H`：查看当前类的继承层级结构  
- `Ctrl + N`：查找当前项目及所有库文件范围内的文件
  、类等

#### Ctrl + Alt
- `Ctrl + Alt + L`：格式化代码
- `Ctrl + Alt + O`：自动优化导入功能，去除未调用导入库

#### Ctrl + Shift
- `Ctrl + Shift + F`：全局搜索整个项目文件内容（包括变量,代码内容等）, ==需保证在英文输入法下进行,否则会与输入法繁简体变换快捷键冲突==

### 运行/调试及配置
#### 配置
- 多窗口运行：勾选 `Allow multiple instances`
- 传参：`Parameters`，运行代码时，固定终端的传入参数
#### 运行
#### 调试
- 查看断点: `Run -> View Breakpoints`

后端debug

- 通过debug启动服务
- 设置对应服务的断点,通过服务访问进行调试
